Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript. HTML is the building block of the web, allowing developers to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. The HTML document starts with a doctype declaration, followed by the html tag that encompasses the entire document. Inside the html tag, there are two main sections: the head and the body. The head contains meta-information about the document, such as the title, while the body contains the content that will be displayed on the web page.
